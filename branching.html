Branching

Definition:

Branches are the result of a statement within jQuery which is determined by the condition of an If/Else function

Here is an example of an If/Else function:

$(document).ready(function() {
        var age = parseInt(prompt("How old are you?"));

        if (age >= 21) {
          $('#drinks').show();
        } else {
          $('#under-21').show();
        }
      });
