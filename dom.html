DOM (Document Object Model) Manipulation

Definition:

The Document Object Model is a tree diagram representation of the hierarchy of html elements. Manipulation of the DOM is acheived by using javascript methods to change and/or output html. Methods like .prepend() and .append() are able to output values created through javascript to the html.

The following javascript method appends a message to a paragraph tag in the html:

javascript

$("p").append(" happy");

html

<p>The dog is</p>

result:

The dog is happy.
