CSS

Definition:

CSS stands for cascading style sheet, which is code that allows formatting of various aspects of html pages such as text and tables. CSS style sheets can be applied to multiple html pages.

For example, the following CSS code will make the text red in any element with a class of "color-red":

  • in html:

    • <p class=".color-red"> "hello" </p>

  • in css:

    • .color-red { color: red;}

    Result:

    hello
