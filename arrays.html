Arrays

Definition:

An array is a list of things grouped together. This group is identified with [ ] brackets. Arrays can be assigned to variables for use in javascript.

Here's an example of an array of numbers:

[3,22,90,12,5]

Here's an example of adding two arrays:

      var one = [1,3];
      var two = [3, 4];
      one+two
      "1,33,4"
